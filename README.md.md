# 🔐 Password Generator

## 📌 Description
A Python-based password generator that creates strong and secure passwords based on user preferences.

## 🚀 Features
- Custom password length
- Option to include numbers
- Option to include symbols
- Random and secure password generation

## 🛠 Technologies Used
- Python
- random module
- string module

## ▶️ How to Run
1. Open terminal
2. Run:
   python main.py
