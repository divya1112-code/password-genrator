# 🔐 Password Generator
## 📌 Description
A Python-based password generator that creates strong and secure passwords based on user preferences such as length, inclusion of numbers, and symbols.  
The project is also deployed as a web application using Streamlit for an interactive user experience.
## 🚀 Features
- Custom password length  
- Option to include numbers  
- Option to include symbols  
- Random and secure password generation  
- Password strength checker (Weak / Medium / Strong)  
- Web-based interface using Streamlit

- string module  
- Streamlit  
## ▶️ How to Run Locally
1. Open terminal  
2. Install Streamlit:
   pip install streamlit  
3. Run:
   streamlit run app.py
